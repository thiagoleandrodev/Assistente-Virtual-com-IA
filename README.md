# Edu — Assistente Financeiro Pessoal

Projeto pessoal para desenvolver o Edu, um agente de IA que ajuda o cliente
a entender seus próprios gastos com base no histórico de transações.

## Estrutura

- [`AGENT.md`](AGENT.md) — quem o Edu é, persona, comportamento e limites (fonte única de verdade).
- [`knowledge.md`](knowledge.md) — o que o Edu sabe sobre os dados que consulta.
- [`data/`](data/) — arquivos de dados que o Edu usa (transações, etc.).
- [`src/app.py`](src/app.py) — aplicação que roda o Edu.

Este repositório é o "motor" do agente — depois de pronto, o código de
`src/` é levado para o lab `dio-lab-bia-do-futuro` como a aplicação funcional
do desafio do Bootcamp Bradesco.
